Projects

DotaDubs

DotaDubs is a predictive AI tool that uses machine learning to forecast the outcome of 1v1 esports matches. Developed with a supervised learning algorithm, DotaDubs achieves 66% accuracy, surpassing the industry baseline by 16%. This project involved building and deploying an interactive Streamlit application to showcase the model's capabilities.

Women in Data Datathon

This project, which won 2nd place in the Women in Data Datathon (out of 200+ teams!), analyzed global heatwave monitoring systems. Using historical weather data and identifying gaps in current systems, we pinpointed the top 8 countries where improved heatwave monitoring would have the greatest impact.
